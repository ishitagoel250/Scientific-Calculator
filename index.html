BASIC MODE

The Basic Mode of the Scientific Calculator consists of the following functions:
Addition(+), Subtraction(-), Multiplication(*), Division(/), Square(^2) and percentage(%)






ADVANCED MODE

The Advanced Mode of the Scientific Calculator consists of the following functions:
sin, cos, tan, X^, log, ln, e, rad, root (X^1/2), factorial (X!) and pi
